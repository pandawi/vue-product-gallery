# Vue Product Gallery

In order to use a mock API with express server, please go to `./client/services/api.service.js`

To start:

```bash
$ npm install
```

To develop:

```bash
$ npm start
```

To build for production:

```bash
$ npm run build
```

To lint you code:

```bash
$ npm run lint
```
